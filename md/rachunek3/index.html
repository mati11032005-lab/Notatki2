    3.Różniczkowanie funkcji

    I. Definicja i Interpretacja Geometryczna Pochodnej¶

    Pochodna funkcji \(f(x)\) w punkcie \(x_0\), oznaczana jako \(\boldsymbol{f'(x_0)}\), jest granicą ilorazu różnicowego. Definicja ta pozwala nam obliczyć chwilową prędkość zmiany funkcji.

    Definicja¶

    Pochodna jest definiowana jako: \(\(f'(x_0) = \lim_{h \to 0} \frac{f(x_0 + h) - f(x_0)}{h}\)\)

    Interpretacja Geometryczna¶

    • Pochodna \(f'(x_0)\) to współczynnik kierunkowy prostej stycznej do wykresu funkcji w punkcie \((x_0, f(x_0))\). Mówi nam o stromości wykresu w tym miejscu.
    • Monotoniczność:
      • Jeśli \(\boldsymbol{f'(x) > 0}\), funkcja rośnie.
      • Jeśli \(\boldsymbol{f'(x) < 0}\), funkcja maleje.
      • Jeśli \(\boldsymbol{f'(x) = 0}\), funkcja ma punkt stacjonarny (styczna jest pozioma).

    Zadanie i Rozwiązanie (1.3)¶

    Zadanie: Oblicz pochodną funkcji \(f(x) = x^2\) w punkcie \(x=3\), korzystając z definicji.

    Rozwiązanie: 1. Obliczenie licznika: \(f(3 + h) - f(3) = (3+h)^2 - 3^2 = (9 + 6h + h^2) - 9 = 6h + h^2\). 2. Iloraz różnicowy: \(\frac{6h + h^2}{h} = \frac{h(6 + h)}{h} = 6 + h\). 3. Obliczenie granicy: \(f'(3) = \lim_{h \to 0} (6 + h) = 6\). 4. Wynik: Pochodna \(f'(3) = 6\).

    II. Reguły Różniczkowania¶

    Aby przyspieszyć obliczenia, używamy reguł pochodnych, bazując na pochodnych podstawowych funkcji (np. \((\boldsymbol{x^n})' = nx^{n-1}\), \((\boldsymbol{e^x})' = e^x\), \((\boldsymbol{\sin x})' = \cos x\)).

    Główne Reguły¶

    1. Pochodna Sumy/Różnicy: \((f(x) \pm g(x))' = f'(x) \pm g'(x)\).
    2. Pochodna Iloczynu: \((f(x)g(x))' = f'(x)g(x) + f(x)g'(x)\).
    3. Pochodna Ilorazu: \(\left(\frac{f(x)}{g(x)}\right)' = \frac{f'(x)g(x) - f(x)g'(x)}{[g(x)]^2}\).
    4. Reguła Łańcuchowa: Dla funkcji złożonej \(f(x) = g(h(x))\), pochodna wynosi: \(f'(x) = g'(h(x)) \cdot h'(x)\) (pochodna funkcji zewnętrznej razy pochodna funkcji wewnętrznej).

    Zadanie i Rozwiązanie (Reguły Ilorazu i Łańcuchowa)¶

    Zadanie: Oblicz pochodną funkcji \(f(x) = \frac{\sin(2x)}{e^x}\).

    Rozwiązanie: 1. Identyfikacja: Licznik \(f(x) = \sin(2x)\), Mianownik \(g(x) = e^x\). 2. Pochodne składowe: * \(f'(x) = (\sin(2x))' = \cos(2x) \cdot 2 = 2\cos(2x)\) (Reguła Łańcuchowa). * \(g'(x) = (e^x)' = e^x\). 3. Zastosowanie Reguły Ilorazu: \(\(f'(x) = \frac{(2\cos(2x)) \cdot e^x - \sin(2x) \cdot e^x}{(e^x)^2}\)\) 4. Uproszczenie: Wyciągamy \(e^x\) z licznika i skracamy: \(\(f'(x) = \frac{e^x(2\cos(2x) - \sin(2x))}{e^{2x}} = \frac{2\cos(2x) - \sin(2x)}{e^x}\)\) 5. Wynik: \(\boldsymbol{f'(x) = e^{-x}(2\cos(2x) - \sin(2x))}\).

    III. Zastosowanie: Ekstrema Lokalna i Krzywizna¶

    Pochodne wyższych rzędów (druga pochodna \(\boldsymbol{f''(x)}\)) opisują kształt wykresu i pomagają klasyfikować punkty stacjonarne.

    1. Druga Pochodna i Krzywizna¶

    • Druga pochodna to pochodna pierwszej pochodnej (\(f''(x) = (f'(x))'\)). Mówi nam, jak zmienia się nachylenie wykresu.
    • Wklęsłość i Wypukłość:
      • Jeśli \(\boldsymbol{f''(x) > 0}\), funkcja jest wypukła (krzywizna w górę, jak uśmiech).
      • Jeśli \(\boldsymbol{f''(x) < 0}\), funkcja jest wklęsła (krzywizna w dół, jak smutna buzia).
    • Punkt Przegięcia: Punkt, w którym zmienia się znak \(f''(x)\) (a więc zmienia się krzywizna). Warunek konieczny: \(f''(x)=0\).

    2. Warunki Istnienia Ekstremów Lokalnych¶

    Aby znaleźć i sklasyfikować ekstrema (maksima i minima), używamy dwóch kroków:

    • Warunek Konieczny (Krok 1): Znajdź punkty stacjonarne, gdzie \(\boldsymbol{f'(x) = 0}\).
    • Warunek Wystarczający (Krok 2 - Test Drugiej Pochodnej):
      • Jeśli \(f'(x_0)=0\) i \(\boldsymbol{f''(x_0) > 0}\) (wypukłość), funkcja ma MINIMUM lokalne.
      • Jeśli \(f'(x_0)=0\) i \(\boldsymbol{f''(x_0) < 0}\) (wklęsłość), funkcja ma MAKSIMUM lokalne.

    Zadanie: Pełna Analiza Funkcji \(f(x) = x^3 - 3x\)¶

    Zadanie: Przeprowadź pełną analizę funkcji \(f(x) = x^3 - 3x\) (monotoniczność, ekstrema, krzywizna).

    Rozwiązanie: 1. Pierwsza pochodna i punkty stacjonarne: * \(f'(x) = 3x^2 - 3\). * \(f'(x) = 0 \implies 3x^2 - 3 = 0 \implies x^2 = 1 \implies \boldsymbol{x = -1}\) i \(\boldsymbol{x = 1}\). 2. Monotoniczność i Ekstrema (Analiza znaku \(f'(x)\)): * \(f'(x) > 0\) dla \(x \in (-\infty, -1) \cup (1, \infty)\) (Funkcja rośnie). * \(f'(x) < 0\) dla \(x \in (-1, 1)\) (Funkcja maleje). * Zmiana znaku z + na - w \(x=-1\) to MAKSIMUM LOKALNE. * Zmiana znaku z - na + w \(x=1\) to MINIMUM LOKALNE. 3. Druga pochodna i Punkt Przegięcia: * \(f''(x) = (3x^2 - 3)' = \boldsymbol{6x}\). * \(f''(x) = 0 \implies 6x = 0 \implies \boldsymbol{x = 0}\) (Punkt Przegięcia). 4. Krzywizna: * Dla \(x < 0\), \(f''(x) < 0\) (Wklęsłość). * Dla \(x > 0\), \(f''(x) > 0\) (Wypukłość).

    IV. Zastosowanie: Problemy Optymalizacyjne¶

    Pochodne są kluczowe w optymalizacji, pozwalając znaleźć maksymalne lub minimalne wartości (np. największy zysk, najmniejszy koszt).

    Zadanie: Maksymalizacja Pola¶

    Zadanie: Mamy \(100 \text{ m}\) ogrodzenia. Jakie wymiary prostokątnego ogródka dadzą największe pole?

    Rozwiązanie: 1. Definicja funkcji (Pole): Niech wymiary to \(x\) i \(y\). Obwód: \(2x + 2y = 100 \implies y = 50 - x\). * Funkcja Pola: \(\boldsymbol{P(x) = x(50 - x) = 50x - x^2}\). 2. Pochodna: \(P'(x) = 50 - 2x\). 3. Punkt Stacjonarny: \(P'(x) = 0 \implies 50 - 2x = 0 \implies \boldsymbol{x = 25}\). 4. Weryfikacja: Druga pochodna \(P''(x) = -2\) (jest ujemna, co gwarantuje maksimum). 5. Wynik: \(x = 25 \text{ m}\). Wymiar \(y = 50 - 25 = 25 \text{ m}\). * Wymiary to \(\mathbf{25 \text{ m} \times 25 \text{ m}}\) (kwadrat).
