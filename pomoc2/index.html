    1) \(f(x) = x - 2 \ln x\)

    • Dziedzina: \(x \in (0, +\infty)\).

    • Pierwsza pochodna i punkty stacjonarne:

    \(f'(x) = 1 - \frac{2}{x} = \frac{x}{x} - \frac{2}{x} = \mathbf{\frac{x-2}{x}}\).

    \(f'(x) = 0 \implies\) Ułamek równa się zero tylko wtedy, gdy jego licznik równa się zero.

    Wyjaśnienie:

    \(x - 2 = 0\) \(\mathbf{x = 2}\).

    • Monotoniczność i Ekstrema:

    \(f'(x) < 0\) dla \(x \in (0, 2)\) (Funkcja maleje).

    \(f'(x) > 0\) dla \(x \in (2, +\infty)\) (Funkcja rośnie).

    Zmiana znaku z - na + w \(x=2\) to MINIMUM LOKALNE.

    • Druga pochodna i Punkt Przegięcia:

    \(f''(x) = (1 - 2x^{-1})' = -2(-1)x^{-2} = \mathbf{\frac{2}{x^2}}\).

    \(f''(x) = 0 \implies\) Licznik musiałby być zerem.

    Wyjaśnienie:

    \(2 = 0\) \(\to\) Sprzeczność.

    Dlatego brak punktów przegięcia.

    • Krzywizna:

    Licznik (2) jest dodatni, mianownik (\(x^2\)) jest dodatni.

    Wynik zawsze \(>0\).

    Funkcja jest wypukła (\(\cup\)) w całej dziedzinie.

    2) \(f(x) = x^2 \ln x\)

    • Dziedzina: \(x \in (0, +\infty)\).

    • Pierwsza pochodna i punkty stacjonarne:

    \(f'(x) = 2x \ln x + x^2 \cdot \frac{1}{x} = 2x \ln x + x = \mathbf{x(2 \ln x + 1)}\).

    \(f'(x) = 0 \implies\) Iloczyn jest zerem, gdy jeden z czynników jest zerem.

    Wyjaśnienie:

    \(x = 0\) (Odrzucamy, bo nie należy do dziedziny \(x>0\)).

    \(2 \ln x + 1 = 0 \implies 2 \ln x = -1 \implies \ln x = -\frac{1}{2}\).

    Aby pozbyć się logarytmu, przykładamy \(e\): \(x = e^{-1/2}\).

    \(\mathbf{x = \frac{1}{\sqrt{e}}}\).

    • Monotoniczność i Ekstrema:

    \(f'(x) < 0\) dla \(x \in (0, \frac{1}{\sqrt{e}})\) (Funkcja maleje).

    \(f'(x) > 0\) dla \(x \in (\frac{1}{\sqrt{e}}, +\infty)\) (Funkcja rośnie).

    Zmiana znaku z - na + w \(x = \frac{1}{\sqrt{e}}\) to MINIMUM LOKALNE.

    • Druga pochodna i Punkt Przegięcia:

    \(f''(x) = (2x \ln x + x)' = (2 \ln x + 2x \cdot \frac{1}{x}) + 1 = 2 \ln x + 2 + 1 = \mathbf{2 \ln x + 3}\).

    \(f''(x) = 0 \implies\) Rozwiązujemy równanie logarytmiczne.

    Wyjaśnienie:

    \(2 \ln x = -3 \quad | :2\)

    \(\ln x = -\frac{3}{2}\).

    \(x = e^{-3/2}\).

    \(\mathbf{x = \frac{1}{\sqrt{e^3}}}\) (Punkt Przegięcia).

    • Krzywizna:

    Dla \(x < e^{-3/2}\) wklęsła (\(\cap\)), dla \(x > e^{-3/2}\) wypukła (\(\cup\)).

    3) \(f(x) = \frac{1+\ln x}{x}\)

    • Dziedzina: \(x \in (0, +\infty)\).

    • Pierwsza pochodna i punkty stacjonarne:

    \(f'(x) = \frac{(\frac{1}{x}) \cdot x - (1+\ln x) \cdot 1}{x^2} = \frac{1 - 1 - \ln x}{x^2} = \mathbf{\frac{-\ln x}{x^2}}\).

    \(f'(x) = 0 \implies\) Licznik ułamka musi być zerem.

    Wyjaśnienie:

    \(-\ln x = 0 \implies \ln x = 0\).

    Logarytm wynosi zero tylko dla argumentu 1 (bo \(e^0 = 1\)).

    \(\mathbf{x = 1}\).

    • Monotoniczność i Ekstrema:

    \(f'(x) > 0\) dla \(x \in (0, 1)\) (bo \(-\ln(ułamek)\) jest dodatni). Funkcja rośnie.

    \(f'(x) < 0\) dla \(x \in (1, +\infty)\). Funkcja maleje.

    Zmiana znaku z + na - w \(x=1\) to MAKSIMUM LOKALNE.

    • Druga pochodna i Punkt Przegięcia:

    \(f''(x) = \frac{(-\frac{1}{x})x^2 - (-\ln x)(2x)}{x^4} = \frac{-x + 2x \ln x}{x^4} = \frac{x(2 \ln x - 1)}{x^4} = \mathbf{\frac{2 \ln x - 1}{x^3}}\).

    \(f''(x) = 0 \implies\) Licznik musi być zerem.

    Wyjaśnienie:

    \(2 \ln x - 1 = 0 \implies 2 \ln x = 1 \implies \ln x = \frac{1}{2}\).

    \(x = e^{1/2}\).

    \(\mathbf{x = \sqrt{e}}\) (Punkt Przegięcia).

    • Krzywizna:

    Dla \(x < \sqrt{e}\) wklęsła (\(\cap\)), dla \(x > \sqrt{e}\) wypukła (\(\cup\)).

    4) \(f(x) = \frac{\ln x}{\sqrt{x}}\)

    • Dziedzina: \(x \in (0, +\infty)\).

    • Pierwsza pochodna i punkty stacjonarne:

    \(f'(x) = \frac{\frac{1}{x}\sqrt{x} - \ln x \cdot \frac{1}{2\sqrt{x}}}{x} = \frac{\frac{2 - \ln x}{2\sqrt{x}}}{x} = \mathbf{\frac{2 - \ln x}{2x\sqrt{x}}}\).

    \(f'(x) = 0 \implies\) Licznik ułamka musi być zerem.

    Wyjaśnienie:

    \(2 - \ln x = 0 \implies 2 = \ln x\).

    Zamieniamy na postać wykładniczą:\(\mathbf{x = e^2}\).

    • Monotoniczność i Ekstrema:

    \(f'(x) > 0\) dla \(x \in (0, e^2)\) (Funkcja rośnie).

    \(f'(x) < 0\) dla \(x \in (e^2, +\infty)\) (Funkcja maleje).

    Zmiana znaku z + na - w \(x=e^2\) to MAKSIMUM LOKALNE.

    • Druga pochodna i Punkt Przegięcia:

    \(f''(x) = \dots\) (obliczenia z reguły ilorazu) \(= \mathbf{\frac{3 \ln x - 8}{4x^2\sqrt{x}}}\).

    \(f''(x) = 0 \implies\) Licznik musi być zerem.

    Wyjaśnienie:

    \(3 \ln x - 8 = 0 \implies 3 \ln x = 8 \implies \ln x = \frac{8}{3}\).

    \(\mathbf{x = e^{8/3}}\) (Punkt Przegięcia).

    • Krzywizna:

    Dla \(x < e^{8/3}\) wklęsła (\(\cap\)), dla \(x > e^{8/3}\) wypukła (\(\cup\)).

    5) \(f(x) = (\ln x)^3 - 3 \ln x\)

    • Dziedzina: \(x \in (0, +\infty)\).

    • Pierwsza pochodna i punkty stacjonarne:

    \(f'(x) = 3(\ln x)^2 \cdot \frac{1}{x} - \frac{3}{x} = \frac{3}{x} (\ln^2 x - 1)\).

    \(f'(x) = 0 \implies\) Ułamek jest zerem, gdy nawias w liczniku jest zerem.

    Wyjaśnienie:

    \(\ln^2 x - 1 = 0 \implies \ln^2 x = 1\).

    Mamy dwie opcje: \(\ln x = 1\) LUB \(\ln x = -1\).

    \(\mathbf{x = e}\) oraz \(\mathbf{x = e^{-1} = \frac{1}{e}}\).

    • Monotoniczność i Ekstrema:

    Analizujemy znak wyrażenia \(\ln^2 x - 1\) (jak paraboli \(t^2 - 1\)):

    Funkcja rośnie w \((0, e^{-1}) \cup (e, +\infty)\).

    Funkcja maleje w \((e^{-1}, e)\).

    \(x = e^{-1}\) to MAKSIMUM, \(x = e\) to MINIMUM.

    • Druga pochodna i Punkt Przegięcia:

    \(f''(x) = \mathbf{\frac{6\ln x - 3\ln^2 x + 3}{x^2}}\)

    \(f''(x) = 0 \implies\) Licznik równa się zero.

    Wyjaśnienie:

    \(-3\ln^2 x + 6\ln x + 3 = 0 \quad | :(-3)\)

    \(\ln^2 x - 2\ln x - 1 = 0\). Podstawiamy \(t = \ln x\).

    \(t^2 - 2t - 1 = 0\). Obliczamy \(\Delta = 4 - 4(1)(-1) = 8\).

    \(t_1 = \frac{2 - \sqrt{8}}{2} = 1 - \sqrt{2}, \quad t_2 = 1 + \sqrt{2}\).

    Wracamy do \(x\):

    \(\mathbf{x = e^{1-\sqrt{2}}}\) oraz \(\mathbf{x = e^{1+\sqrt{2}}}\) (Punkty Przegięcia).

    • Krzywizna:

    Wklęsła na zewnątrz punktów przegięcia, wypukła pomiędzy nimi (zgodnie ze znakiem paraboli w liczniku pomnożonej przez \(-3\)).

    6) \(f(x) = \frac{1}{\ln x}\)

    • Dziedzina: \(x \in (0, 1) \cup (1, +\infty)\).

    Wyjaśnienie: \(x>0\) (z logarytmu) ORAZ \(\ln x \neq 0 \implies x \neq 1\) (z mianownika).

    • Pierwsza pochodna i punkty stacjonarne:

    \(f'(x) = -1(\ln x)^{-2} \cdot \frac{1}{x} = \mathbf{\frac{-1}{x (\ln x)^2}}\).

    \(f'(x) = 0 \implies\) Licznik musi być zerem.

    Wyjaśnienie:

    \(-1 = 0\).To jest sprzeczność. B rak punktów stacjonarnych (brak ekstremów).

    • Monotoniczność i Ekstrema:

    Licznik (-1) jest ujemny. Mianowniki \(x\) i \((\ln x)^2\) są dodatnie w dziedzinie.

    Całość jest zawsze ujemna. Funkcja stale maleje (w obu przedziałach dziedziny).

    • Druga pochodna i Punkt Przegięcia:

    \(f''(x) = \mathbf{\frac{\ln x + 2}{x^2 (\ln x)^3}}\)

    \(f''(x) = 0 \implies\) Licznik musi być zerem.

    Wyjaśnienie:

    \(\ln x + 2 = 0 \implies \ln x = -2\).

    \(\mathbf{x = e^{-2} = \frac{1}{e^2}}\) (Punkt Przegięcia).

    • Krzywizna:

    Analizujemy znak licznika i mianownika:

    Dla \(x \in (0, e^{-2})\): Wypukła (\(\cup\)).

    Dla \(x \in (e^{-2}, 1)\): Wklęsła (\(\cap\)).

    Dla \(x \in (1, +\infty)\): Wypukła (\(\cup\)).

    7) \(f(x) = e^{-x^2}\) (Krzywa Gaussa)

    • Dziedzina: \(x \in \mathbb{R}\).

    • Pierwsza pochodna i punkty stacjonarne:

    \(f'(x) = -2x e^{-x^2}\).

    \(f'(x) = 0 \implies -2x \cdot \underbrace{e^{-x^2}}_{\neq 0} = 0\).

    Wyjaśnienie: Ponieważ \(e^{-x^2}\) nigdy nie jest zerem, możemy podzielić równanie przez to wyrażenie (lub po prostu je pominąć). Zostaje tylko:

    \(-2x = 0 \quad | :(-2)\)

    \(\mathbf{x = 0}\).

    • Monotoniczność i Ekstrema:

    \(f'(x) > 0\) dla \(x \in (-\infty, 0)\) (bo np. dla \(x=-1\): \(-2(-1) > 0\)).

    \(f'(x) < 0\) dla \(x \in (0, +\infty)\) (bo np. dla \(x=1\): \(-2(1) < 0\)).

    Zmiana znaku z + na - w \(x=0\) to MAKSIMUM LOKALNE.

    • Druga pochodna i Punkt Przegięcia:

    \(f''(x) = 2e^{-x^2}(2x^2 - 1)\).

    \(f''(x) = 0 \implies (2x^2 - 1) \cdot \underbrace{2e^{-x^2}}_{\neq 0} = 0\).

    Wyjaśnienie: Zostaje nam tylko nawias.

    \(2x^2 - 1 = 0 \implies 2x^2 = 1 \quad |:2\)

    \(x^2 = \frac{1}{2} \implies x = \sqrt{\frac{1}{2}}\) lub \(x = -\sqrt{\frac{1}{2}}\).

    \(\mathbf{x = \frac{1}{\sqrt{2}}}\) oraz \(\mathbf{x = -\frac{1}{\sqrt{2}}}\) (Punkty Przegięcia).

    • Krzywizna:

    Analiza znaku \(2x^2-1\) (parabola ramionami w górę): Wypukła na zewnątrz pierwiastków, wklęsła pomiędzy nimi.

    8) \(f(x) = x e^{-x^2/2}\)

    • Dziedzina: \(x \in \mathbb{R}\).

    • Pierwsza pochodna i punkty stacjonarne:

    \(f'(x) = e^{-x^2/2}(1-x^2)\).

    \(f'(x) = 0 \implies (1-x^2) \cdot \underbrace{e^{-x^2/2}}_{\neq 0} = 0\).

    Wyjaśnienie: Przyrównujemy nawias do zera.

    \(1 - x^2 = 0 \implies 1 = x^2\).

    \(\mathbf{x = 1}\) oraz \(\mathbf{x = -1}\).

    • Monotoniczność i Ekstrema:

    \(x = -1\) (MIN), \(x = 1\) (MAX) – wynikają z wykresu paraboli \(1-x^2\) (ramiona w dół).

    • Druga pochodna i Punkt Przegięcia:

    \(f''(x) = x e^{-x^2/2}(x^2 - 3)\).

    \(f''(x) = 0 \implies x(x^2 - 3) \cdot \underbrace{e^{-x^2/2}}_{\neq 0} = 0\).

    Wyjaśnienie: Mamy iloczyn dwóch czynników, który ma dać zero. Albo \(x=0\), albo nawias \((x^2-3)=0\).

    \(\mathbf{x = 0}\).

    \(x^2 - 3 = 0 \implies x^2 = 3 \implies \mathbf{x = \sqrt{3}}\) lub \(\mathbf{x = -\sqrt{3}}\).

    • Krzywizna:

    Trzy punkty przegięcia dzielą oś na 4 przedziały.

    9) \(f(x) = x^3 e^{-4x}\)

    • Dziedzina: \(x \in \mathbb{R}\).

    • Pierwsza pochodna i punkty stacjonarne:

    \(f'(x) = x^2 e^{-4x}(3 - 4x)\).

    \(f'(x) = 0 \implies x^2 (3 - 4x) \cdot \underbrace{e^{-4x}}_{\neq 0} = 0\).

    Wyjaśnienie: Zostaje nam iloczyn \(x^2\) i nawiasu.

    \(x^2 = 0 \implies \mathbf{x = 0}\).

    \(3 - 4x = 0 \implies 3 = 4x \implies \mathbf{x = \frac{3}{4}}\).

    • Monotoniczność i Ekstrema:

    W punkcie \(x=0\) mamy \(x^2\), co oznacza, że znak pochodnej się nie zmienia (jest dodatni przed i po zerze). Dlatego w \(x=0\) nie ma ekstremum.

    Zmiana znaku następuje tylko w \(x = \frac{3}{4}\) (z plusa na minus \(\to\) MAX).

    • Druga pochodna i Punkt Przegięcia:

    \(f''(x) = 2x e^{-4x}(8x^2 - 12x + 3)\).

    \(f''(x) = 0 \implies 2x(8x^2 - 12x + 3) = 0\).

    Wyjaśnienie: Mamy dwa przypadki:

    \(2x = 0 \implies \mathbf{x = 0}\).

    \(8x^2 - 12x + 3 = 0\). Obliczamy \(\Delta\):

    \(\Delta = (-12)^2 - 4 \cdot 8 \cdot 3 = 144 - 96 = 48\).

    \(\sqrt{\Delta} = \sqrt{16 \cdot 3} = 4\sqrt{3}\).

    \(x_1 = \frac{12 - 4\sqrt{3}}{16} = \frac{3-\sqrt{3}}{4}, \quad x_2 = \frac{3+\sqrt{3}}{4}\).

    Mamy więc trzy punkty przegięcia: \(0\) oraz dwa wyliczone z delty.

    10) \(f(x) = e^{-x^2+8x-14}\)

    • Dziedzina: \(x \in \mathbb{R}\).

    • Pierwsza pochodna i punkty stacjonarne:

    \(f'(x) = e^{-x^2+8x-14} \cdot (-2x+8)\).

    \(f'(x) = 0 \implies (-2x + 8) \cdot \underbrace{e^{\dots}}_{\neq 0} = 0\).

    Wyjaśnienie: Ignorujemy \(e\). Zostaje nawias.

    \(-2x + 8 = 0 \implies 8 = 2x \quad | :2\)

    \(\mathbf{x = 4}\).

    • Monotoniczność i Ekstrema:

    Dla \(x < 4\) (np. \(x=0\)) mamy \(-2(0)+8 = 8 > 0\) (Rośnie).

    Dla \(x > 4\) (np. \(x=5\)) mamy \(-2(5)+8 = -2 < 0\) (Maleje).

    \(x=4\) to MAKSIMUM LOKALNE.

    • Druga pochodna i Punkt Przegięcia:

    \(f''(x) = e^{(\dots)} [(-2x+8)^2 - 2]\).

    \(f''(x) = 0 \implies ( -2x+8 )^2 - 2 = 0\).

    Wyjaśnienie:

    \((2x - 8)^2 = 2\). Pierwiastkujemy obie strony:

    \(|2x - 8| = \sqrt{2}\).

    Opcja A: \(2x - 8 = \sqrt{2} \implies 2x = 8 + \sqrt{2} \implies \mathbf{x = 4 + \frac{\sqrt{2}}{2}}\).

    Opcja B: \(2x - 8 = -\sqrt{2} \implies 2x = 8 - \sqrt{2} \implies \mathbf{x = 4 - \frac{\sqrt{2}}{2}}\).

    11) \(f(x) = \ln(1+e^{-x})\)

    • Dziedzina: \(x \in \mathbb{R}\).

    • Pierwsza pochodna i punkty stacjonarne:

    \(f'(x) = \frac{-e^{-x}}{1+e^{-x}}\).

    \(f'(x) = 0 \implies\) ułamek równa się zero tylko wtedy, gdy jego licznik równa się zero.

    Wyjaśnienie:

    \(-e^{-x} = 0 \implies e^{-x} = 0\).

    To równanie jest sprzeczne (funkcja wykładnicza nigdy nie osiąga zera). Dlatego brak rozwiązań i brak ekstremów.

    • Monotoniczność i Ekstrema:

    Licznik (\(-e^{-x}\)) jest zawsze ujemny (bo \(e^{-x}\) dodatnie, a stoi minus). Mianownik (\(1+e^{-x}\)) jest zawsze dodatni.

    Wynik (minus przez plus) daje minus. Funkcja stale maleje.

    • Druga pochodna i Punkt Przegięcia:

    \(f''(x) = \frac{e^{-x}}{(1+e^{-x})^2}\).

    \(f''(x) = 0 \implies e^{-x} = 0\).

    Znowu sprzeczność. Licznik nigdy nie będzie zerem. Licznik dodatni, mianownik (kwadrat) dodatni \(\to\) Wynik zawsze dodatni. Funkcja wypukła.

    12) \(f(x) = e^{\text{arctg } x}\)

    • Dziedzina: \(x \in \mathbb{R}\).

    • Pierwsza pochodna i punkty stacjonarne:

    \(f'(x) = e^{\text{arctg } x} \cdot \frac{1}{1+x^2}\).

    \(f'(x) = 0 \implies\) Licznik ułamka musi być zerem.

    Wyjaśnienie: Licznikiem jest \(1 \cdot e^{\text{arctg } x}\).

    \(e^{\text{arctg } x} = 0\) \(\to\) Sprzeczność. Funkcja nigdy się nie zeruje.

    • Monotoniczność i Ekstrema:

    Wszystkie elementy (\(e^{\dots}\), \(1\), \(1+x^2\)) są dodatnie. Pochodna zawsze \(>0\). Funkcja stale rośnie.

    • Druga pochodna i Punkt Przegięcia:

    \(f''(x) = e^{\text{arctg } x} \frac{1-2x}{(1+x^2)^2}\).

    \(f''(x) = 0 \implies\) Ułamek jest zerem, gdy licznik jest zerem.

    Wyjaśnienie: Ignorujemy \(e^{\dots}\) (jest dodatnie). Patrzymy tylko na licznik ułamka:

    \(1 - 2x = 0\).

    \(1 = 2x \quad | :2\)

    \(\mathbf{x = \frac{1}{2}}\) (Punkt Przegięcia).
