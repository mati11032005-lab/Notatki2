    I. Rozgrzewka (Potęgi i Pierwiastki)

    Główny wzór: \((x^n)' = n x^{n-1}\).

    Pamiętaj, że \(\sqrt[n]{x^m} = x^{\frac{m}{n}}\) oraz \(\frac{1}{x^n} = x^{-n}\).

    1.\(y = \frac{1}{3}x^3 - \frac{3}{2}x^4 + \frac{13}{5}x^5 - 2x^6\)

    Rozwiązanie:

    \[y' = \frac{1}{3} \cdot 3x^2 - \frac{3}{2} \cdot 4x^3 + \frac{13}{5} \cdot 5x^4 - 2 \cdot 6x^5\]
    \[y' = x^2 - 6x^3 + 13x^4 - 12x^5\]

    2.\(y = \frac{4}{x^3}\)

    Rozwiązanie:

    Przekształcenie: \(y = 4x^{-3}\)

    \[y' = 4 \cdot (-3)x^{-4} = -12x^{-4} = \frac{-12}{x^4}\]

    3.\(y = 3x^{7/3} - 4x^{13/4} + \frac{4}{7}x^{-1/2}\)

    Rozwiązanie:

    \[y' = 3 \cdot \frac{7}{3}x^{4/3} - 4 \cdot \frac{13}{4}x^{9/4} + \frac{4}{7} \cdot \left(-\frac{1}{2}\right)x^{-3/2}\]
    \[y' = 7x^{4/3} - 13x^{9/4} - \frac{2}{7}x^{-3/2}\]

    4.\(y = \sqrt{x} - 5\sqrt[3]{x^2} - 2\sqrt{x^3}\) Rozwiązanie: Przekształcenie na potęgi: \(y = x^{1/2} - 5x^{2/3} - 2x^{3/2}\)

    \[y' = \frac{1}{2}x^{-1/2} - 5 \cdot \frac{2}{3}x^{-1/3} - 2 \cdot \frac{3}{2}x^{1/2}\]

    \(\(y' = \frac{1}{2\sqrt{x}} - \frac{10}{3\sqrt[3]{x}} - 3\sqrt{x}\)\) 5. \(y = \frac{2}{x^3\sqrt{x}}\) Rozwiązanie: Przekształcenie: \(x^3 \cdot x^{1/2} = x^{3.5} = x^{7/2}\). Zatem \(y = 2x^{-7/2}\).

    \(\(y' = 2 \cdot \left(-\frac{7}{2}\right)x^{-9/2} = -7x^{-9/2} = \frac{-7}{x^4\sqrt{x}}\)\) 6. \(x = t^3\sqrt{t}\) Rozwiązanie: Przekształcenie: \(x = t^3 \cdot t^{1/2} = t^{3.5} = t^{7/2}\).

    \(\(x' = \frac{7}{2}t^{5/2} = \frac{7}{2}t^2\sqrt{t}\)\) 7. \(y = (4x^2-2x\sqrt{x}+x)(2x+\sqrt{x})\) Rozwiązanie: Komentarz: Zamiast stosować wzór na pochodną iloczynu (który tu byłby długi), najpierw wymnóż nawiasy. \(y = 8x^3 + 4x^2\sqrt{x} - 4x^2\sqrt{x} - 2x^2 + 2x^2 + x\sqrt{x}\) Zauważ, że wyrazy się skracają! Zostaje: \(y = 8x^3 + x^{3/2}\).

    \(\(y' = 24x^2 + \frac{3}{2}x^{1/2} = 24x^2 + \frac{3}{2}\sqrt{x}\)\) II. Ułamki i Funkcja Złożona Wzór na iloraz: \((\frac{f}{g})' = \frac{f'g - fg'}{g^2}\). Wzór na złożenie: \((f(g))' = f'(g) \cdot g'\). 8. \(y = \frac{x+1}{x-1}\) Rozwiązanie:

    \(\(y' = \frac{(1)(x-1) - (x+1)(1)}{(x-1)^2} = \frac{x-1-x-1}{(x-1)^2} = \frac{-2}{(x-1)^2}\)\) 9. \(y = \frac{5x^2+x-2}{x^2+7}\) Rozwiązanie:

    \[y' = \frac{(10x+1)(x^2+7) - (5x^2+x-2)(2x)}{(x^2+7)^2}\]

    \(\(y' = \frac{10x^3+70x+x^2+7 - (10x^3+2x^2-4x)}{(x^2+7)^2} = \frac{-x^2+74x+7}{(x^2+7)^2}\)\) 10. \(y = \frac{3}{(1-x^2)(1-2x^3)}\) Rozwiązanie: Komentarz: Można wymnożyć mianownik, ale zróbmy to jako \((3u^{-1})'\), gdzie \(u\) to mianownik. Lub standardowo ilorazem (licznik to stała, więc jego pochodna to 0). \(y = 3 \cdot [(1-x^2)(1-2x^3)]^{-1}\). Skorzystajmy ze wzoru \((\frac{1}{v})' = \frac{-v'}{v^2}\). Niech \(v = (1-x^2)(1-2x^3)\). \(v' = (-2x)(1-2x^3) + (1-x^2)(-6x^2) = -2x+4x^4 -6x^2+6x^4 = 10x^4-6x^2-2x\).

    \(\(y' = 3 \cdot \frac{-(10x^4-6x^2-2x)}{[(1-x^2)(1-2x^3)]^2} = \frac{-6x(5x^3-3x-1)}{(1-x^2)^2(1-2x^3)^2}\)\) 11. \(s = (3t+1)^7\) Rozwiązanie (Funkcja złożona): Pochodna zewnętrzna (potęga) razy pochodna wewnętrzna (wnętrze nawiasu).

    \(\(s' = 7(3t+1)^6 \cdot (3) = 21(3t+1)^6\)\) 12. \(y = \sqrt{x^2-4}\) Rozwiązanie: Traktujemy to jako \((x^2-4)^{1/2}\).

    \(\(y' = \frac{1}{2\sqrt{x^2-4}} \cdot (2x) = \frac{x}{\sqrt{x^2-4}}\)\) 13. \(y = \frac{1}{\sqrt{2-3t}}\) Rozwiązanie: Przekształć: \(y = (2-3t)^{-1/2}\).

    \(\(y' = -\frac{1}{2}(2-3t)^{-3/2} \cdot (-3) = \frac{3}{2}(2-3t)^{-3/2} = \frac{3}{2\sqrt{(2-3t)^3}}\)\) 14. \(y = \frac{3\sqrt{x}}{x^2+1}\) Rozwiązanie:

    \(\(y' = \frac{(3 \cdot \frac{1}{2\sqrt{x}})(x^2+1) - (3\sqrt{x})(2x)}{(x^2+1)^2}\)\) Mnożymy licznik i mianownik przez \(2\sqrt{x}\), żeby usunąć piętrowy ułamek:

    \(\(y' = \frac{3(x^2+1) - 12x^2}{2\sqrt{x}(x^2+1)^2} = \frac{3 - 9x^2}{2\sqrt{x}(x^2+1)^2}\)\) 15. \(y = \sqrt[4]{(x-1)^3}\) Rozwiązanie: \(y = (x-1)^{3/4}\).

    \(\(y' = \frac{3}{4}(x-1)^{-1/4} = \frac{3}{4\sqrt[4]{x-1}}\)\) III. Trygonometria i Funkcje Wykładnicze Pamiętaj: \((\sin x)' = \cos x\), \((\cos x)' = -\sin x\), \((e^x)' = e^x\). 16. \(y = e^{3x}\) Rozwiązanie:

    \(\(y' = e^{3x} \cdot (3x)' = 3e^{3x}\)\) 17. \(y = e^{\sin x}\) Rozwiązanie:

    \(\(y' = e^{\sin x} \cdot (\sin x)' = e^{\sin x} \cos x\)\) 18. \(y = 5e^{\cos x}\) Rozwiązanie:

    \(\(y' = 5e^{\cos x} \cdot (-\sin x) = -5\sin x e^{\cos x}\)\) 19. \(y = x^2 e^{2x} \sin x\) Rozwiązanie (Potrójny iloczyn): Wzór: \((abc)' = a'bc + ab'c + abc'\).

    \(\(y' = (2x)e^{2x}\sin x + x^2(2e^{2x})\sin x + x^2 e^{2x}(\cos x)\)\) Wyciągamy \(xe^{2x}\) przed nawias:

    \(\(y' = xe^{2x}(2\sin x + 2x\sin x + x\cos x)\)\) 20. \(z = 2x+\sin 2x\) Rozwiązanie:

    \(\(z' = 2 + \cos(2x) \cdot 2 = 2(1+\cos 2x)\)\) 21. \(s = \sin^2 3t\) Rozwiązanie: To jest \((\sin(3t))^2\). Mamy trzy warstwy: kwadrat, sinus, \(3t\).

    \[s' = 2\sin 3t \cdot (\sin 3t)' = 2\sin 3t \cdot \cos 3t \cdot 3\]

    \(\(s' = 6\sin 3t \cos 3t = 3\sin 6t\)\) (ze wzoru na sinus podwojonego kąta). 22. \(v = \frac{5}{\sin^3 2t}\) Rozwiązanie: \(v = 5(\sin 2t)^{-3}\).

    \[v' = 5 \cdot (-3)(\sin 2t)^{-4} \cdot (\cos 2t) \cdot 2\]

    \(\(v' = -30 \frac{\cos 2t}{\sin^4 2t} = -30 \frac{\text{ctg } 2t}{\sin^3 2t}\)\) 23. \(y = \frac{x \sin x}{1+\text{tg } x}\) Rozwiązanie: Licznik: \((x\sin x)' = 1\cdot\sin x + x\cos x\). Mianownik: \((1+\text{tg } x)' = \frac{1}{\cos^2 x}\).

    \(\(y' = \frac{(\sin x + x\cos x)(1+\text{tg } x) - x\sin x \cdot \frac{1}{\cos^2 x}}{(1+\text{tg } x)^2}\)\) (Można to dalej upraszczać trygonometrycznie, ale ta postać jest poprawną pochodną). 24. \(y = \cos x - \frac{1}{3}\cos^3 x\) Rozwiązanie:

    \[y' = -\sin x - \frac{1}{3} \cdot 3\cos^2 x \cdot (-\sin x)\]

    \(\(y' = -\sin x + \sin x \cos^2 x = -\sin x (1-\cos^2 x) = -\sin x (\sin^2 x) = -\sin^3 x\)\) IV. Logarytmy Wzór: \((\ln x)' = \frac{1}{x}\). 25. \(y = \ln 3x\) Rozwiązanie:

    \(\(y' = \frac{1}{3x} \cdot 3 = \frac{1}{x}\)\) Komentarz: Zauważ, że \(\ln 3x = \ln 3 + \ln x\), pochodna stałej to 0, więc zostaje tylko \(1/x\). 26. \(y = 5\ln 10x\) Rozwiązanie:

    \(\(y' = 5 \cdot \frac{1}{10x} \cdot 10 = \frac{5}{x}\)\) 27. \(y = \ln \sin x\) Rozwiązanie:

    \(\(y' = \frac{1}{\sin x} \cdot \cos x = \text{ctg } x\)\) 28. \(y = \ln(1+\frac{a}{x})\) Rozwiązanie: Funkcja wewnętrzna to \(1 + ax^{-1}\), jej pochodna to \(-ax^{-2}\).

    \(\(y' = \frac{1}{1+\frac{a}{x}} \cdot \left(\frac{-a}{x^2}\right) = \frac{x}{x+a} \cdot \frac{-a}{x^2} = \frac{-a}{x(x+a)}\)\) 29. \(y = 3^x x^3\) Rozwiązanie: Iloczyn. \((a^x)' = a^x \ln a\).

    \(\(y' = (3^x \ln 3) \cdot x^3 + 3^x \cdot (3x^2) = 3^x x^2 (x\ln 3 + 3)\)\) 30. \(y = 5 \cdot 10^{3x}\) Rozwiązanie:

    \(\(y' = 5 \cdot 10^{3x} \ln 10 \cdot (3x)' = 15 \cdot 10^{3x} \ln 10\)\) V. Waga Ciężka (Zaawansowane) 31. \(u = \frac{\sqrt{1+v}-\sqrt{1-v}}{\sqrt{1+v}+\sqrt{1-v}}\) Rozwiązanie: Komentarz: Nie licz pochodnej od razu! Najpierw usuń niewymierność z mianownika (pomnóż licznik i mianownik przez licznik). \(u = \frac{(\sqrt{1+v}-\sqrt{1-v})^2}{(1+v)-(1-v)} = \frac{1+v - 2\sqrt{1-v^2} + 1-v}{2v} = \frac{2-2\sqrt{1-v^2}}{2v} = \frac{1-\sqrt{1-v^2}}{v}\) Teraz liczmy pochodną z tej prostszej formy:

    \[u' = \frac{-( \frac{-2v}{2\sqrt{1-v^2}} ) \cdot v - (1-\sqrt{1-v^2}) \cdot 1}{v^2}\]

    \(\(u' = \frac{\frac{v^2}{\sqrt{1-v^2}} - 1 + \sqrt{1-v^2}}{v^2} = \frac{v^2 - \sqrt{1-v^2} + 1 - v^2}{v^2\sqrt{1-v^2}} = \frac{1-\sqrt{1-v^2}}{v^2\sqrt{1-v^2}}\)\) 32. \(y = \sqrt{\frac{a-x}{a^2-x^2}}\) Rozwiązanie: Uproszczenie: \(a^2-x^2 = (a-x)(a+x)\). \(y = \sqrt{\frac{a-x}{(a-x)(a+x)}} = \sqrt{\frac{1}{a+x}} = (a+x)^{-1/2}\)

    \(\(y' = -\frac{1}{2}(a+x)^{-3/2} = \frac{-1}{2\sqrt{(a+x)^3}}\)\) 33. \(y = \sqrt{\sin x + \sqrt{x+2\sqrt{x}}}\) Rozwiązanie: Stosujemy regułę łańcuchową "cebulkową" (od zewnątrz do wewnątrz). Pierwiastek zewnętrzny: \(\frac{1}{2\sqrt{\dots}}\) Pochodna wnętrza: \(\cos x + (\text{pochodna pierwiastka średniego})\) Pochodna średniego: \(\frac{1}{2\sqrt{\dots}} \cdot (1 + \text{pochodna małego})\) Pochodna małego: \(2 \cdot \frac{1}{2\sqrt{x}} = \frac{1}{\sqrt{x}}\)

    \(\(y' = \frac{1}{2\sqrt{\sin x + \sqrt{x+2\sqrt{x}}}} \cdot \left( \cos x + \frac{1}{2\sqrt{x+2\sqrt{x}}} \cdot \left( 1 + \frac{1}{\sqrt{x}} \right) \right)\)\) 34. \(y = \ln \sqrt{\frac{1+\sin x}{1-\sin x}}\) Rozwiązanie: Komentarz: Koniecznie użyj własności logarytmu! \(\ln A^n = n\ln A\) oraz \(\ln(A/B) = \ln A - \ln B\). \(y = \frac{1}{2} [\ln(1+\sin x) - \ln(1-\sin x)]\)

    \[y' = \frac{1}{2} \left[ \frac{\cos x}{1+\sin x} - \frac{-\cos x}{1-\sin x} \right] = \frac{\cos x}{2} \left[ \frac{1-\sin x + 1 + \sin x}{1-\sin^2 x} \right]\]

    \(\(y' = \frac{\cos x}{2} \cdot \frac{2}{\cos^2 x} = \frac{1}{\cos x}\)\) 35. \(y = \ln(e^{mx}+e^{-mx})\) Rozwiązanie:

    \(\(y' = \frac{1}{e^{mx}+e^{-mx}} \cdot (me^{mx} - me^{-mx}) = m \frac{e^{mx}-e^{-mx}}{e^{mx}+e^{-mx}} = m \cdot \text{tgh}(mx)\)\) 36. \(y = x^{5x}\) Rozwiązanie (Pochodna Logarytmiczna): Nie działa wzór \(x^n\) ani \(a^x\). Sposób: \(y = e^{\ln(x^{5x})} = e^{5x \ln x}\).

    \[y' = e^{5x \ln x} \cdot (5x \ln x)'\]

    \(\(y' = x^{5x} \cdot (5\ln x + 5x \cdot \frac{1}{x}) = 5x^{5x}(\ln x + 1)\)\) 37. \(y = x^{\sin x}\) Rozwiązanie: Zapisz jako \(y = e^{\sin x \ln x}\).

    \[y' = x^{\sin x} \cdot (\sin x \ln x)'\]

    \(\(y' = x^{\sin x} \left( \cos x \ln x + \sin x \cdot \frac{1}{x} \right)\)\) 38. \(y = (\text{tg } x)^{\sin x}\) Rozwiązanie: \(y = e^{\sin x \ln(\text{tg } x)}\).

    \(\(y' = (\text{tg } x)^{\sin x} \cdot \left[ \cos x \ln(\text{tg } x) + \sin x \cdot \frac{1}{\text{tg } x} \cdot \frac{1}{\cos^2 x} \right]\)\) Upraszczamy drugi człon w nawiasie: \(\sin x \cdot \frac{\cos x}{\sin x} \cdot \frac{1}{\cos^2 x} = \frac{1}{\cos x}\).

    \(\(y' = (\text{tg } x)^{\sin x} \left( \cos x \ln(\text{tg } x) + \frac{1}{\cos x} \right)\)\) 39. \(y = x^{e^x}\) Rozwiązanie: \(y = e^{e^x \ln x}\).

    \(\(y' = x^{e^x} \cdot (e^x \ln x)' = x^{e^x} \left( e^x \ln x + e^x \cdot \frac{1}{x} \right) = x^{e^x} e^x \left(\ln x + \frac{1}{x}\right)\)\) 40. \(y = x^{x^x}\) Rozwiązanie: To "wieża". Zapiszmy: \(y = e^{x^x \ln x}\). Potrzebujemy pochodnej wykładnika \(u = x^x \ln x\). Wiemy z przykładu (lub uczymy się na pamięć), że \((x^x)' = x^x(\ln x + 1)\). Zatem:

    \(\(u' = (x^x)' \ln x + x^x (\ln x)' = x^x(\ln x + 1)\ln x + x^x \frac{1}{x}\)\) Wracamy do \(y\):

    \[y' = x^{x^x} \cdot \left[ x^x(\ln x + 1)\ln x + x^{x-1} \right]\]
